# SOC Daily Lab

Laboratorio diario de SOC Analyst sobre Wazuh (SIEM/XDR), emulando el flujo real de un puesto L1 → L2: generar actividad, detectarla, hacer triage y documentar.

Cada entrada en `fase-N/` corresponde a una jornada: objetivo, consulta, análisis y lecciones. Stack: Wazuh 4.14.6 (Docker) · Kali (atacante) · Ubuntu y macOS (endpoints monitoreados).
